# Laundry Shop Website

A modern laundry service website built using HTML and CSS. This project is designed as a landing page for a laundry business, showcasing services, pricing, and contact information in a clean and user-friendly layout.

## 🚀 Features

- Attractive homepage design
- Services section
- Pricing / offers section
- Contact section
- Clean and simple UI
- Responsive layout (if added)

## 🛠️ Built With

- HTML5
- CSS3
